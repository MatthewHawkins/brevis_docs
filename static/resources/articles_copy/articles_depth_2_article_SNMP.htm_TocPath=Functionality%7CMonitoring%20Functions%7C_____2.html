SNMP

 

Configuration

 

 

i You will find the SNMP configuration in the web interface under Configuration - General.

 

 

 

You can activate the SNMP service and set the SNMP Community String.

 

The SNMP Community String only supports characters, numbers and minus ( a-z A-Z 0-9 - ).

 

With the SNMP Whitelist you can permit the access to the SNMP Service for IPs or subnets.

 

 

Default settings

 

The default SNMP Community String is public and the SNMP service is deactivated.

 

The SNMP Whitelist is empty on default, means all (0.0.0.0/0) IPs or subnets have access.

 

 

SNMP Manager
