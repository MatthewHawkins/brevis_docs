Groups

 

 

 

ê For older devices without support, this function requires the licensing of the Receiver groups module.

 

 

 

i The options for this function are located in the web interface under Configuration - Groups.

 

 

 

 

 

Column Description
ID

Consecutive number of the group

Instead of the name, you can specify the ID of the group using HTTP API.
Name

Name of the group

The name may not contain any accents, blanks or special characters (except for dot, hyphen und underscore).
Members Contacts from the address book who are members of this group

 

 

➯Use the buttons beneath the table to create new groups or import contacts and groups.
