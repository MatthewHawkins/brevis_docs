Upload license

 

 

 

i An import for license files is provided in the web interface under Configuration - Upload license.

 

 

 

 

  • Klick on Choose file and choose the license file on your computer (*.lic).

 

  • Klick the Upload Button.

 

 

 

ê You just can upload license-files (*.lic).
